数据加密

拾光忆栈提供强大的加密功能，保护您的敏感记忆和个人数据安全。本指南将介绍如何设置和使用加密功能。

加密概述​

拾光忆栈的加密功能包括：

  • 端到端加密：数据在本地加密，只有您知道解密密钥
  • 选择性加密：可以只加密敏感记忆，保持其他记忆易于访问
  • 传输加密：同步过程中的数据传输加密
  • 存储加密：本地存储和云存储中的数据加密

加密保护的内容包括：

  • 记忆内容（文本、图片、视频等）
  • 记忆元数据（标题、标签、分类等）
  • 用户设置和偏好（可选）
  • 搜索索引（可选）

加密类型​

拾光忆栈提供多种加密类型：

全局加密​

全局加密会加密所有记忆和数据：

  1. 所有数据使用同一个主密钥加密
  2. 提供最高级别的安全性
  3. 需要在每次使用应用时输入密码
  4. 可能会影响搜索和过滤功能

选择性加密​

选择性加密允许您只加密敏感记忆：

  1. 只有标记为"敏感"的记忆会被加密
  2. 其他记忆保持未加密状态
  3. 平衡了安全性和便利性
  4. 允许在不输入密码的情况下访问非敏感记忆

分类加密​

分类加密允许您加密特定分类下的所有记忆：

  1. 为不同分类设置不同的加密密钥
  2. 适合将记忆分为不同安全级别
  3. 可以与不同人共享不同分类的访问权限
  4. 提供更精细的访问控制

设置加密​

启用全局加密​

设置全局加密的步骤：

  1. 打开设置页面
  2. 找到"安全设置"部分
  3. 点击"配置加密"按钮
  4. 选择"全局加密"
  5. 设置主密码：
    • 输入强密码
    • 确认密码
    • 设置密码提示（可选）
  6. 选择加密算法和强度
  7. 点击"启用加密"按钮
  8. 等待加密过程完成

重要：请妥善保管您的主密码。如果忘记密码，将无法恢复加密的数据。

启用选择性加密​

设置选择性加密的步骤：

  1. 打开设置页面
  2. 找到"安全设置"部分
  3. 点击"配置加密"按钮
  4. 选择"选择性加密"
  5. 设置加密密码
  6. 选择加密算法和强度
  7. 点击"启用加密"按钮

启用后，您可以标记特定记忆为"敏感"，这些记忆将被加密。

启用分类加密​

设置分类加密的步骤：

  1. 打开设置页面
  2. 找到"安全设置"部分
  3. 点击"配置加密"按钮
  4. 选择"分类加密"
  5. 选择要加密的分类
  6. 为每个分类设置加密密码
  7. 选择加密算法和强度
  8. 点击"启用加密"按钮

使用加密​

加密记忆​

在选择性加密模式下，标记记忆为敏感：

  1. 打开记忆详情页面
  2. 点击"更多"菜单
  3. 选择"标记为敏感"
  4. 确认操作

系统会使用您的加密密钥加密该记忆。

查看加密记忆​

查看加密记忆的步骤：

  1. 点击加密记忆
  2. 系统会提示输入密码
  3. 输入正确密码后，记忆会被解密并显示
  4. 记忆在会话期间保持解密状态，或根据设置在一段时间后重新加密

批量加密​

批量加密多个记忆：

  1. 在浏览记忆页面，选择多个记忆
  2. 点击"更多"菜单
  3. 选择"批量加密"
  4. 输入加密密码
  5. 确认操作

更改加密设置​

更改加密设置的步骤：

  1. 打开设置页面
  2. 找到"安全设置"部分
  3. 点击"更改加密设置"按钮
  4. 输入当前密码
  5. 选择要更改的设置：
    • 更改密码
    • 更改加密算法
    • 更改加密范围
  6. 设置新选项
  7. 点击"应用更改"按钮

注意：更改加密设置需要重新加密所有数据，可能需要一些时间。

加密算法​

拾光忆栈支持多种加密算法：

AES-256​

高级加密标准（AES）是最广泛使用的对称加密算法：

  • 256位密钥提供极高的安全性
  • 快速且资源消耗低
  • 适合大多数用户的默认选择

ChaCha20-Poly1305​

现代流密码，在某些平台上比AES更快：

  • 高性能加密和认证
  • 在移动设备上效率更高
  • 提供与AES相当的安全性

Twofish​

AES的替代方案，提供额外的安全边际：

  • 128位块密码
  • 可变密钥长度
  • 没有已知的有效攻击

密钥管理​

密码管理​

管理加密密码的最佳实践：

  1. 使用强密码（至少16个字符，包含大小写字母、数字和特殊字符）
  2. 不要在多个服务中重复使用相同的密码
  3. 考虑使用密码管理器存储密码
  4. 定期更改密码（每3-6个月）

密钥备份​

备份加密密钥的方法：

  1. 在设置页面，找到"安全设置"部分
  2. 点击"导出加密密钥"按钮
  3. 输入当前密码
  4. 选择导出格式：
    • 加密文件（需要额外密码）
    • 纸质备份（打印密钥恢复码）
  5. 将备份存储在安全位置

密钥恢复​

使用备份恢复密钥的步骤：

  1. 在登录页面点击"忘记密码"
  2. 选择恢复方式：
    • 使用备份文件
    • 输入恢复码
  3. 按照提示完成恢复过程
  4. 设置新密码

高级加密功能​

双因素解密​

启用双因素解密增强安全性：

  1. 在安全设置中启用"双因素解密"
  2. 选择第二因素类型：
    • 移动设备验证
    • 硬件安全密钥
    • 生物识别（如果设备支持）
  3. 完成设置过程

启用后，解密数据需要密码和第二因素。

紧急访问​

设置紧急访问允许受信任的人在紧急情况下访问您的数据：

  1. 在安全设置中找到"紧急访问"部分
  2. 点击"添加紧急联系人"
  3. 输入联系人信息
  4. 设置等待期和访问级别
  5. 生成恢复密钥并安全地分享给联系人

联系人需要遵循特定流程并等待设定的等待期才能访问您的数据。

安全擦除​

配置安全擦除功能：

  1. 在安全设置中找到"安全擦除"部分
  2. 设置触发条件：
    • 多次密码错误
    • 特定手势或命令
    • 定时条件
  3. 选择擦除范围：
    • 仅加密数据
    • 所有数据
    • 完全重置应用
  4. 保存设置

警告：安全擦除会永久删除数据，无法恢复。请谨慎使用此功能。

加密与搜索​

加密会影响搜索功能，拾光忆栈提供以下解决方案：

加密搜索索引​

允许搜索加密内容：

  1. 在安全设置中启用"加密搜索索引"
  2. 选择索引加密级别
  3. 了解搜索性能可能会受到影响

部分索引​

只索引非敏感内容：

  1. 在安全设置中选择"部分索引"
  2. 选择要索引的内容类型
  3. 加密内容将不被索引，需要手动浏览

解密后搜索​

临时解密内容进行搜索：

  1. 输入主密码解锁加密内容
  2. 在解锁状态下执行搜索
  3. 搜索结果包括加密和未加密内容
  4. 会话结束后重新加密

加密与同步​

加密与多设备同步的结合：

加密同步​

确保同步过程中的数据安全：

  1. 数据在上传前在本地加密
  2. 云存储中只存储加密数据
  3. 下载后在本地解密
  4. 传输过程使用TLS/SSL加密

密钥同步​

在多设备间安全地同步密钥：

  1. 在主设备上导出加密密钥
  2. 在新设备上导入密钥
  3. 或使用密码派生函数在每个设备上独立生成相同的密钥

同步冲突​

处理加密数据的同步冲突：

  1. 需要先解密才能解决冲突
  2. 解决冲突后重新加密
  3. 上传解决后的加密数据

合规性和法律考虑​

数据保护法规​

拾光忆栈的加密功能有助于遵守数据保护法规：

  • 符合GDPR对个人数据保护的要求
  • 支持CCPA规定的数据安全措施
  • 帮助满足HIPAA等行业特定法规的要求

法律限制​

了解您所在地区对加密的法律限制：

  • 某些国家/地区可能限制使用加密技术
  • 可能需要向当局提供加密密钥
  • 使用前请了解当地法律

故障排除​

忘记密码​

如果忘记加密密码：

  1. 尝试使用密码提示回忆密码
  2. 使用之前设置的密钥备份恢复
  3. 如果设置了紧急访问，联系您的紧急联系人
  4. 如果以上方法都不可行，可能需要重置应用并丢失加密数据

加密错误​

如果遇到加密或解密错误：

  1. 确保输入了正确的密码
  2. 检查应用版本是否最新
  3. 尝试重启应用
  4. 检查设备存储空间是否充足
  5. 如果问题持续，尝试恢复备份

如果问题仍然存在，请参阅故障排除章节或联系我们的支持团队。
