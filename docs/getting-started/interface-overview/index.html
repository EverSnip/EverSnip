界面概览

拾光忆栈的界面设计直观且功能丰富，本指南将帮助您了解各个界面组件及其功能，让您能够高效地使用拾光忆栈管理您的记忆。

主界面布局​

拾光忆栈的主界面由以下几个主要部分组成：

  1. 顶部工具栏 - 包含主要功能按钮和搜索框
  2. 侧边导航栏 - 提供主要功能区域的导航
  3. 内容区域 - 显示当前选择的功能页面
  4. 状态栏 - 显示同步状态、存储使用情况等信息

弹出窗口​

点击Chrome工具栏中的拾光忆栈图标会打开弹出窗口，提供快速访问常用功能：

  1. 快速保存 - 保存当前页面或选中内容
  2. 最近记忆 - 显示最近添加的记忆
  3. 快速搜索 - 搜索您的记忆库
  4. 快捷功能 - 访问常用功能的快捷按钮
  5. 设置入口 - 打开设置页面

导航栏​

侧边导航栏提供对主要功能区域的访问：

  1. 仪表盘 - 显示概览和统计信息
  2. 浏览记忆 - 查看和管理已保存的记忆
  3. 添加记忆 - 手动添加新记忆
  4. 分类管理 - 管理记忆分类结构
  5. 标签管理 - 管理记忆标签
  6. 设置 - 访问应用设置
  7. 帮助与支持 - 获取帮助和支持

仪表盘页面​

仪表盘提供您的记忆库概览和使用统计：

  1. 统计卡片 - 显示记忆总数、分类数、标签数等统计
  2. 最近活动 - 显示最近的添加、编辑和查看活动
  3. 存储使用情况 - 显示本地和云存储使用情况
  4. 快速访问 - 常用分类和标签的快速访问
  5. 记忆趋势 - 显示记忆添加趋势图表

浏览记忆页面​

浏览记忆页面是查看和管理记忆的中心：

  1. 搜索栏 - 搜索记忆内容
  2. 过滤器 - 按分类、标签、日期等过滤记忆
  3. 视图切换 - 在列表、卡片和其他视图之间切换
  4. 排序选项 - 更改记忆的排序方式
  5. 记忆列表/网格 - 显示记忆的主要区域
  6. 记忆详情 - 显示选中记忆的详细内容
  7. 批量操作 - 对多个记忆执行操作

添加记忆页面​

添加记忆页面用于创建新的记忆条目。

  1. 记忆编辑器

    • 标题输入框
    • 内容编辑区域（支持富文本）
    • 分类选择器
    • 标签输入框
    • 媒体上传区域（图片、视频）

  2. 来源信息

    • 当前页面URL（自动填充）
    • 来源网站名称（自动填充）

  3. 操作按钮

    • 保存 - 保存记忆
    • 取消 - 取消编辑
    • 预览 - 预览记忆效果

设置页面​

设置页面用于配置拾光忆栈的各项功能和选项。

  1. 用户信息设置

    • 用户资料
    • 账户信息

  2. 分类管理

    • 添加/编辑/删除分类
    • 设置默认分类

  3. 外观设置

    • 主题模式（亮色/暗色）
    • 主题颜色
    • 字体大小
    • 布局选项

  4. 高级设置

    • 数据加密选项
    • 自动锁定设置
    • 性能优化选项

  5. 同步设置

    • 云存储配置
    • 同步频率
    • 冲突解决策略

  6. 导入导出

    • 数据备份
    • 数据恢复
    • 从其他服务导入

上下文菜单​

右键点击网页内容时，拾光忆栈会在浏览器的上下文菜单中添加选项：

  1. 保存到拾光忆栈 - 保存选中内容
  2. 保存图片到拾光忆栈 - 保存右键点击的图片
  3. 保存链接到拾光忆栈 - 保存右键点击的链接
  4. 保存视频到拾光忆栈 - 保存右键点击的视频
  5. 截取页面到拾光忆栈 - 保存页面截图

通知​

拾光忆栈会在特定操作后显示通知：

  1. 保存成功 - 成功保存记忆后显示
  2. 同步完成 - 同步操作完成后显示
  3. 错误通知 - 操作失败时显示错误信息
  4. 更新通知 - 应用更新后显示新功能信息

键盘快捷键​

拾光忆栈提供多种键盘快捷键提高使用效率：

  • Ctrl+Shift+M（Windows/Linux）或 Cmd+Shift+M（Mac）- 保存选中内容
  • Ctrl+Shift+E（Windows/Linux）或 Cmd+Shift+E（Mac）- 编辑并保存选中内容
  • Ctrl+Shift+P（Windows/Linux）或 Cmd+Shift+P（Mac）- 打开扩展弹出窗口
  • Ctrl+Shift+F（Windows/Linux）或 Cmd+Shift+F（Mac）- 搜索记忆
  • Ctrl+Shift+S（Windows/Linux）或 Cmd+Shift+S（Mac）- 截取页面

您可以在设置页面自定义这些快捷键。

移动适配​

拾光忆栈的界面会根据窗口大小自动调整：

  1. 紧凑导航 - 在小屏幕上导航栏会折叠为菜单按钮
  2. 单列布局 - 内容区域会调整为单列布局
  3. 触摸优化 - 按钮和控件会变大以适应触摸操作
  4. 手势支持 - 支持滑动和其他触摸手势

主题和外观​

拾光忆栈支持多种主题和外观选项：

  1. 亮色主题 - 明亮背景，适合日间使用
  2. 暗色主题 - 深色背景，减少夜间使用的眼睛疲劳
  3. 自定义主题 - 自定义颜色和样式
  4. 高对比度模式 - 提高可访问性的高对比度显示

扩展集成​

拾光忆栈可以与其他浏览器功能和扩展集成：

  1. 书签集成 - 可以将记忆添加为浏览器书签
  2. 阅读模式集成 - 与浏览器的阅读模式协同工作
  3. 分享功能 - 与浏览器的分享功能集成
  4. 下载管理 - 与浏览器的下载管理器集成

界面自定义​

您可以根据个人偏好自定义拾光忆栈的界面：

  1. 在设置页面，找到"外观设置"部分
  2. 调整布局、颜色、字体等选项
  3. 选择显示或隐藏特定界面元素
  4. 自定义工具栏按钮
  5. 设置默认视图和排序方式

辅助功能​

拾光忆栈提供多种辅助功能，提高可访问性：

  1. 屏幕阅读器支持 - 兼容屏幕阅读器软件
  2. 键盘导航 - 完全支持键盘导航
  3. 文本缩放 - 支持浏览器文本缩放
  4. 高对比度模式 - 提高文本和背景对比度
  5. 减少动画 - 可以减少或禁用界面动画

状态指示器​

界面中的各种指示器帮助您了解应用状态：

  1. 同步状态 - 显示数据同步状态
  2. 存储使用量 - 显示存储空间使用情况
  3. 网络状态 - 显示网络连接状态
  4. 加密状态 - 显示数据加密状态
  5. 更新指示器 - 显示可用更新

开始使用界面​

如果您是新用户，建议按照以下步骤熟悉界面：

  1. 浏览仪表盘，了解整体情况
  2. 尝试使用弹出窗口保存一条记忆
  3. 在浏览记忆页面查看已保存的记忆
  4. 熟悉不同的视图模式和过滤选项
  5. 探索设置页面，根据个人偏好自定义应用

熟悉界面后，您可以进一步了解保存记忆、浏览记忆等具体功能的使用方法。
