浏览记忆

拾光忆栈提供多种方式浏览和管理您保存的记忆，帮助您快速找到所需信息。本指南将介绍如何使用这些功能高效浏览您的记忆库。

记忆列表​

记忆列表是浏览记忆的主要界面：

列表视图​

默认的列表视图显示记忆的基本信息：

  1. 标题 - 记忆的标题
  2. 内容预览 - 记忆内容的简短预览
  3. 来源 - 记忆的来源网站
  4. 日期 - 保存日期
  5. 标签 - 关联的标签
  6. 操作按钮 - 编辑、删除等操作

卡片视图​

卡片视图以网格形式显示记忆：

  1. 点击工具栏中的"视图"按钮
  2. 选择"卡片视图"
  3. 记忆将以卡片形式显示，突出显示图片和标题

杂志视图​

杂志视图提供更丰富的视觉体验：

  1. 点击"视图"按钮
  2. 选择"杂志视图"
  3. 记忆将以类似杂志的布局显示，大小不一

时间线视图​

时间线视图按时间顺序组织记忆：

  1. 点击"视图"按钮
  2. 选择"时间线视图"
  3. 记忆将按日期分组显示在时间线上

排序和过滤​

排序记忆​

您可以按不同条件排序记忆：

  1. 点击工具栏中的"排序"按钮
  2. 选择排序条件：
    • 最新添加 - 按添加时间从新到旧排序
    • 最早添加 - 按添加时间从旧到新排序
    • 标题 A-Z - 按标题字母顺序排序
    • 标题 Z-A - 按标题字母逆序排序
    • 来源 - 按来源网站排序
    • 最近查看 - 按最近查看时间排序

过滤记忆​

使用过滤器缩小记忆范围：

  1. 使用左侧边栏中的过滤选项：

    • 分类 - 按分类过滤
    • 标签 - 按标签过滤
    • 日期范围 - 按保存日期过滤
    • 内容类型 - 按类型（文本、图片、视频等）过滤
    • 来源 - 按来源网站过滤

  2. 组合多个过滤条件进行精确过滤

保存过滤器​

保存常用过滤器以便快速访问：

  1. 设置所需的过滤条件
  2. 点击过滤面板底部的"保存过滤器"按钮
  3. 输入过滤器名称
  4. 保存的过滤器会显示在"已保存过滤器"部分

记忆详情​

查看记忆详情​

点击记忆列表中的任何记忆可查看其详细内容：

  1. 详情面板会显示完整内容
  2. 包括原始格式、链接和媒体
  3. 显示元数据（标签、分类、来源等）
  4. 提供相关操作按钮

详情面板位置​

您可以调整详情面板的位置：

  1. 在设置中找到"界面设置"
  2. 选择详情面板位置：
    • 右侧 - 详情显示在右侧面板（默认）
    • 底部 - 详情显示在底部面板
    • 弹出窗口 - 详情显示在弹出窗口中

媒体查看​

查看记忆中的媒体内容：

  1. 图片 - 点击可放大，支持缩放和平移
  2. 视频 - 内置播放器，支持常见控制功能
  3. 音频 - 内置播放器，支持播放控制
  4. 文档 - 内置文档查看器

组织记忆​

编辑记忆​

修改已保存的记忆：

  1. 在记忆列表中找到要编辑的记忆
  2. 点击记忆右上角的"编辑"按钮
  3. 在编辑界面修改内容、标题、标签等
  4. 点击"保存"按钮应用更改

移动记忆​

将记忆移动到不同分类：

  1. 选择一个或多个记忆
  2. 点击工具栏中的"移动"按钮
  3. 在弹出的对话框中选择目标分类
  4. 点击"移动"按钮

标记记忆​

使用标记突出显示重要记忆：

  1. 选择一个或多个记忆
  2. 点击工具栏中的"标记"按钮
  3. 选择标记类型：
    • 收藏 - 添加到收藏夹
    • 重要 - 标记为重要
    • 待处理 - 标记为待处理
    • 已归档 - 归档记忆

标记的记忆会显示相应的图标，并可以通过过滤器快速访问。

批量操作​

对多个记忆执行批量操作：

  1. 通过点击记忆前的复选框选择多个记忆
  2. 或使用工具栏中的"全选"按钮
  3. 选择要执行的操作：
    • 批量编辑 - 同时编辑多个记忆的标签、分类等
    • 批量删除 - 删除选中的记忆
    • 批量导出 - 导出选中的记忆
    • 批量标记 - 为选中的记忆添加标记

特殊视图​

收藏夹​

查看收藏的记忆：

  1. 在左侧导航栏中点击"收藏夹"
  2. 显示所有标记为收藏的记忆
  3. 可以按照与普通记忆相同的方式排序和过滤

最近查看​

查看最近访问的记忆：

  1. 在左侧导航栏中点击"最近查看"
  2. 显示最近查看过的记忆，按访问时间排序
  3. 帮助您快速返回之前查看的内容

今日回顾​

查看系统推荐的回顾内容：

  1. 在左侧导航栏中点击"今日回顾"
  2. 系统会根据重要性、查看频率和时间间隔推荐记忆
  3. 帮助您复习重要信息，防止遗忘

高级浏览功能​

全文搜索​

使用搜索功能查找特定内容：

  1. 在顶部搜索框中输入关键词
  2. 按回车键或点击搜索图标
  3. 结果会按相关性排序显示
  4. 匹配的关键词会在结果中高亮显示

详细的搜索功能请参阅搜索与过滤章节。

关联记忆​

查看与当前记忆相关的其他记忆：

  1. 在记忆详情页面，找到"相关记忆"部分
  2. 系统会基于内容相似性、标签和分类显示相关记忆
  3. 点击任何相关记忆可直接查看

记忆图谱​

以可视化方式查看记忆之间的关联：

  1. 点击工具栏中的"记忆图谱"按钮
  2. 系统会显示记忆之间的关联网络
  3. 可以拖动、缩放和重新排列节点
  4. 点击节点可查看相应记忆的详情

键盘导航​

使用键盘快捷键快速浏览记忆：

  • 上/下箭头 - 在记忆列表中上下移动
  • 回车 - 查看选中记忆的详情
  • Esc - 返回记忆列表
  • Ctrl+F (Windows/Linux) 或 Cmd+F (Mac) - 聚焦到搜索框
  • Ctrl+1-9 (Windows/Linux) 或 Cmd+1-9 (Mac) - 快速访问前9个记忆
  • Ctrl+左/右箭头 (Windows/Linux) 或 Cmd+左/右箭头 (Mac) - 在记忆历史中前进/后退

您可以在设置页面自定义这些快捷键。

移动设备浏览​

在移动设备上浏览记忆：

  1. 界面会自动调整为适合小屏幕的布局
  2. 使用滑动手势导航
  3. 点击记忆查看详情
  4. 使用底部导航栏访问主要功能

浏览设置​

自定义浏览体验：

  1. 打开设置页面
  2. 找到"浏览设置"部分
  3. 配置以下选项：
    • 默认视图模式 - 设置默认视图（列表、卡片等）
    • 默认排序 - 设置默认排序方式
    • 每页显示数量 - 设置每页显示的记忆数量
    • 自动加载更多 - 启用/禁用滚动时自动加载更多记忆
    • 记忆预览长度 - 设置列表中显示的预览文本长度

故障排除​

如果您在浏览记忆时遇到问题，请尝试以下解决方法：

记忆不显示​

如果记忆没有正确显示：

  1. 检查是否应用了过滤器
  2. 尝试清除搜索框
  3. 检查分类和标签选择
  4. 刷新页面

加载缓慢​

如果浏览界面加载缓慢：

  1. 减少每页显示的记忆数量
  2. 使用更简单的视图模式（如列表视图）
  3. 关闭自动加载更多功能
  4. 优化存储（删除不需要的记忆或媒体）

如果问题仍然存在，请参阅故障排除章节或联系我们的支持团队。
