分类与标签

拾光忆栈提供分类和标签两种方式组织您的记忆，帮助您创建结构化的知识库。本指南将介绍如何使用这些功能有效组织和管理您的记忆。

分类系统​

分类是一种层级化的组织方式，类似于文件夹结构，适合创建有明确层次的知识体系。

创建分类​

创建新分类的方法：

  1. 打开设置页面
  2. 找到"分类管理"部分
  3. 点击"添加分类"按钮
  4. 输入分类名称
  5. 选择父分类（如果是子分类）
  6. 点击"保存"按钮

编辑分类​

编辑现有分类的方法：

  1. 在分类管理页面找到要编辑的分类
  2. 点击分类旁边的"编辑"图标
  3. 修改分类名称或父分类
  4. 点击"保存"按钮

删除分类​

删除分类的方法：

  1. 在分类管理页面找到要删除的分类
  2. 点击分类旁边的"删除"图标
  3. 在确认对话框中选择如何处理该分类下的记忆：
    • 移动到其他分类
    • 设为未分类
    • 一并删除（谨慎使用）
  4. 点击"确认"按钮

注意：删除分类可能会影响大量记忆。请谨慎操作，并考虑先备份数据。

重新排序分类​

调整分类顺序的方法：

  1. 在分类管理页面，使用拖放功能调整分类顺序
  2. 拖动分类到新位置
  3. 系统会自动保存新的顺序

嵌套分类​

创建层级分类结构的方法：

  1. 创建主分类
  2. 创建子分类，并选择主分类作为父分类
  3. 可以创建多级嵌套结构

例如，您可以创建以下结构：

  • 学习资料
    • 编程
      • Python
      • JavaScript
    • 语言学习
      • 英语
      • 日语

设置默认分类​

设置默认分类的方法：

  1. 打开设置页面
  2. 找到"默认设置"部分
  3. 在"默认分类"下拉菜单中选择一个分类
  4. 保存设置

之后，新保存的记忆会自动分配到默认分类，除非您手动选择其他分类。

标签系统​

标签是一种非层级化的组织方式，允许您为记忆添加多个关键词，适合创建灵活的关联网络。

添加标签​

为记忆添加标签的方法：

  1. 保存或编辑记忆时，在标签字段中输入关键词
  2. 多个标签用逗号分隔
  3. 点击"保存"按钮

创建标签​

拾光忆栈使用动态标签系统，您不需要预先创建标签：

  1. 只需在保存记忆时输入新标签
  2. 系统会自动创建不存在的标签
  3. 输入时会显示现有标签的自动完成建议

管理标签​

管理现有标签的方法：

  1. 打开设置页面
  2. 找到"标签管理"部分
  3. 在这里您可以：
    • 查看所有标签及其使用频率
    • 重命名标签
    • 合并标签
    • 删除标签
    • 创建标签组

重命名标签​

重命名标签的方法：

  1. 在标签管理页面找到要重命名的标签
  2. 点击标签旁边的"编辑"图标
  3. 输入新名称
  4. 点击"保存"按钮

系统会自动更新所有使用该标签的记忆。

合并标签​

合并相似标签的方法：

  1. 在标签管理页面，选择要合并的标签（按住Ctrl或Shift键多选）
  2. 点击"合并标签"按钮
  3. 输入合并后的标签名称
  4. 点击"确认"按钮

例如，您可以将"javascript"、"js"和"JavaScript"合并为统一的"JavaScript"标签。

删除标签​

删除不需要的标签的方法：

  1. 在标签管理页面找到要删除的标签
  2. 点击标签旁边的"删除"图标
  3. 在确认对话框中点击"确认"按钮

删除标签不会删除相关记忆，只会移除标签关联。

标签组​

创建标签组可以更好地组织大量标签：

  1. 在标签管理页面点击"创建标签组"按钮
  2. 输入标签组名称
  3. 选择要包含在组中的标签
  4. 点击"保存"按钮

标签组在浏览和过滤时特别有用，可以一次显示或隐藏一组相关标签。

使用分类和标签浏览记忆​

按分类浏览​

通过分类浏览记忆的方法：

  1. 在浏览记忆页面，点击左侧分类面板中的分类名称
  2. 系统会显示该分类下的所有记忆
  3. 点击带有"+"图标的分类可以展开查看子分类

按标签浏览​

通过标签浏览记忆的方法：

  1. 在浏览记忆页面，点击左侧标签面板中的标签名称
  2. 系统会显示带有该标签的所有记忆
  3. 可以点击多个标签进行组合过滤

组合过滤​

结合分类和标签进行精确过滤：

  1. 在侧边过滤面板中选择一个分类
  2. 再选择一个或多个标签
  3. 系统会显示同时属于所选分类和包含所选标签的记忆

自动标签建议​

拾光忆栈可以根据记忆内容自动建议标签：

启用自动标签​

  1. 打开设置页面
  2. 找到"标签设置"部分
  3. 启用"自动标签建议"选项
  4. 保存设置

使用自动标签​

添加记忆时使用自动标签：

  1. 在添加记忆页面输入内容
  2. 点击"生成标签建议"按钮
  3. 系统会分析内容并建议相关标签
  4. 点击建议的标签添加它们
  5. 您也可以手动添加其他标签

标签统计和分析​

拾光忆栈提供标签使用统计和分析：

查看标签统计​

  1. 打开设置页面
  2. 找到"标签管理"部分
  3. 点击"标签统计"按钮
  4. 查看标签使用频率、增长趋势等统计信息

标签关联分析​

查看标签之间的关联：

  1. 在标签统计页面点击"关联分析"
  2. 系统会显示标签之间的关联强度
  3. 可以发现经常一起使用的标签组合

导入导出分类和标签​

您可以导入导出分类和标签设置：

导出分类和标签​

  1. 打开设置页面
  2. 找到"分类管理"或"标签管理"部分
  3. 点击"导出"按钮
  4. 选择保存位置
  5. 系统会生成一个JSON文件，包含分类或标签信息

导入分类和标签​

  1. 打开设置页面
  2. 找到"分类管理"或"标签管理"部分
  3. 点击"导入"按钮
  4. 选择之前导出的JSON文件
  5. 选择导入选项（合并或替换）
  6. 点击"导入"按钮

这对于在多台设备间同步分类和标签结构特别有用。

最佳实践​

分类最佳实践​

  • 创建清晰的层级结构，避免过深的嵌套（建议不超过3-4层）
  • 使用描述性名称，避免过于宽泛的分类
  • 定期整理和合并相似分类
  • 为常用操作创建特殊分类（如"待处理"、"重要"等）

标签最佳实践​

  • 使用简洁、一致的命名规则（如全部小写或首字母大写）
  • 避免过于相似的标签（如"python"和"python编程"）
  • 使用复合标签表示复杂概念（如"机器学习"而非分开的"机器"和"学习"）
  • 为项目或主题创建前缀标签（如"项目:网站重构"、"学习:Python"）

分类与标签结合使用​

  • 使用分类表示内容的主要类别或领域
  • 使用标签表示内容的具体特征、主题或关键词
  • 避免创建与标签功能重叠的细粒度分类
  • 定期查看标签云，了解您的知识分布

高级技巧​

使用特殊标签​

您可以创建特殊用途的标签：

  • 使用"#"前缀创建项目标签（如"#网站重构"）
  • 使用"@"前缀创建人物标签（如"@团队成员"）
  • 使用"!"前缀创建优先级标签（如"!紧急"）

系统会为这些特殊标签应用不同的样式和过滤选项。

临时标签​

对于临时需求，您可以创建临时标签：

  1. 使用特定前缀（如"tmp:"）创建临时标签
  2. 完成相关任务后，批量删除这些临时标签

标签别名​

为常用标签创建别名：

  1. 在标签管理页面，找到目标标签
  2. 点击"创建别名"按钮
  3. 输入一个或多个别名
  4. 保存设置

之后，使用任何别名都会自动映射到主标签。

故障排除​

分类不显示​

如果分类不正确显示：

  1. 检查分类是否被隐藏（点击"显示所有分类"）
  2. 尝试刷新页面
  3. 检查是否有筛选条件限制了显示

标签不工作​

如果标签功能不正常：

  1. 确保标签输入格式正确（用逗号分隔）
  2. 检查是否达到了每个记忆的标签数量限制
  3. 尝试使用标签管理工具重新整理标签

如果问题仍然存在，请参阅故障排除章节或联系我们的支持团队。
