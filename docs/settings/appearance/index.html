外观设置

外观设置允许您自定义拾光忆栈的视觉样式和界面布局，使其更符合您的个人偏好。本指南将详细介绍如何配置和管理这些设置。

主题设置​

主题模式​

选择拾光忆栈的基本主题模式：

  1. 打开设置页面
  2. 找到"外观设置"部分
  3. 在"主题模式"选项中选择：
    • 亮色模式：使用明亮的背景和深色文本
    • 暗色模式：使用深色背景和浅色文本
    • 跟随系统：自动匹配系统的暗色/亮色模式
    • 自动（时间）：根据时间自动切换模式

主题颜色​

自定义拾光忆栈的主题颜色：

  1. 在外观设置部分找到"主题颜色"选项
  2. 选择预设颜色方案或自定义颜色：
    • 预设方案：选择内置的颜色方案
    • 自定义：点击"自定义"按钮打开颜色选择器
  3. 如果选择自定义，可以设置以下颜色：
    • 主色：主要强调色
    • 次色：次要强调色
    • 成功色：表示成功的颜色
    • 警告色：表示警告的颜色
    • 错误色：表示错误的颜色
  4. 点击"应用"按钮预览效果
  5. 满意后点击"保存"按钮

自定义主题​

创建和管理自定义主题：

  1. 在外观设置部分找到"自定义主题"选项
  2. 点击"创建新主题"按钮
  3. 设置主题名称
  4. 配置主题的所有视觉元素
  5. 点击"保存主题"按钮
  6. 已保存的主题会显示在主题列表中，可以：
    • 应用主题
    • 编辑主题
    • 删除主题
    • 导出/导入主题

界面布局​

布局模式​

选择拾光忆栈的整体布局模式：

  1. 在外观设置部分找到"布局模式"选项
  2. 选择布局模式：
    • 标准布局：传统的三栏布局
    • 紧凑布局：减少空白区域，显示更多内容
    • 宽屏布局：针对宽屏显示器优化
    • 阅读模式：专注于内容阅读的简化布局

侧边栏位置​

设置侧边栏的位置：

  1. 在外观设置部分找到"侧边栏位置"选项
  2. 选择位置：
    • 左侧：侧边栏显示在左侧（默认）
    • 右侧：侧边栏显示在右侧
  3. 设置侧边栏宽度（像素或百分比）
  4. 选择是否允许调整侧边栏大小

工具栏设置​

自定义工具栏的显示和功能：

  1. 在外观设置部分找到"工具栏设置"选项
  2. 配置以下选项：
    • 工具栏位置：顶部或底部
    • 工具栏样式：标准、紧凑或图标
    • 显示的按钮：选择要显示的功能按钮
    • 自定义按钮顺序：拖放调整按钮顺序

标签页设置​

配置标签页的显示方式：

  1. 在外观设置部分找到"标签页设置"选项
  2. 配置以下选项：
    • 标签页位置：顶部或侧边
    • 标签页样式：方形、圆角或胶囊形
    • 标签页行为：点击或悬停激活
    • 显示关闭按钮：始终、悬停时或从不

视觉样式​

字体设置​

自定义文本显示：

  1. 在外观设置部分找到"字体设置"选项
  2. 配置以下选项：
    • 界面字体：应用界面使用的字体
    • 内容字体：记忆内容显示的字体
    • 代码字体：代码块使用的等宽字体
    • 字体大小：基础字体大小（其他元素按比例缩放）

卡片样式​

自定义记忆卡片的外观：

  1. 在外观设置部分找到"卡片样式"选项
  2. 配置以下选项：
    • 卡片形状：方形、圆角或自定义圆角半径
    • 卡片阴影：无、轻微、中等或强烈
    • 卡片边框：无、细线、粗线或自定义
    • 卡片间距：紧凑、标准或宽松
    • 卡片动画：启用/禁用悬停和点击动画

按钮样式​

自定义按钮外观：

  1. 在外观设置部分找到"按钮样式"选项
  2. 配置以下选项：
    • 按钮形状：圆角、方形或胶囊形
    • 按钮大小：小、中、大或自定义
    • 按钮效果：悬停和点击效果
    • 图标按钮：图标按钮的显示方式
    • 按钮组：按钮组的显示样式

表单元素​

自定义表单元素外观：

  1. 在外观设置部分找到"表单元素"选项
  2. 配置以下选项：
    • 输入框样式：输入框的外观
    • 选择器样式：下拉选择器的外观
    • 复选框样式：复选框的外观
    • 单选按钮样式：单选按钮的外观
    • 开关样式：开关控件的外观

高级自定义​

自定义CSS​

添加自定义CSS样式：

  1. 在外观设置部分找到"高级自定义"选项
  2. 点击"自定义CSS"按钮
  3. 在编辑器中输入自定义CSS代码
  4. 点击"应用"按钮预览效果
  5. 满意后点击"保存"按钮

注意：自定义CSS可能会影响应用的稳定性和兼容性。请谨慎使用。

导入导出主题​

分享或备份您的主题设置：

  1. 在高级自定义部分找到"导入导出"选项
  2. 导出主题：
    • 点击"导出主题"按钮
    • 选择导出格式（JSON或主题包）
    • 保存导出文件
  3. 导入主题：
    • 点击"导入主题"按钮
    • 选择主题文件
    • 选择导入选项（完全替换或合并）
    • 点击"导入"按钮

视图设置​

记忆列表视图​

自定义记忆列表的显示方式：

  1. 在外观设置部分找到"记忆列表视图"选项
  2. 配置以下选项：
    • 默认视图模式：列表、卡片、杂志或时间线
    • 列表密度：紧凑、标准或宽松
    • 显示缩略图：始终、仅图片记忆或从不
    • 显示的元数据：选择要显示的元数据字段
    • 排序方式：默认的排序顺序

记忆详情视图​

自定义记忆详情的显示方式：

  1. 在外观设置部分找到"记忆详情视图"选项
  2. 配置以下选项：
    • 详情面板位置：右侧、底部或弹出窗口
    • 详情面板宽度：宽度或比例
    • 显示元数据：选择要显示的元数据
    • 媒体显示：媒体内容的显示方式
    • 代码块样式：代码块的显示样式

编辑器设置​

自定义记忆编辑器的外观：

  1. 在外观设置部分找到"编辑器设置"选项
  2. 配置以下选项：
    • 编辑器主题：亮色或暗色
    • 工具栏位置：顶部或底部
    • 显示的工具：选择要显示的编辑工具
    • 自动保存：启用/禁用自动保存
    • 拼写检查：启用/禁用拼写检查

动画和效果​

动画设置​

自定义界面动画：

  1. 在外观设置部分找到"动画设置"选项
  2. 配置以下选项：
    • 启用动画：全局启用/禁用动画
    • 动画速度：慢、中、快或自定义
    • 过渡效果：淡入淡出、滑动或缩放
    • 页面切换动画：页面之间的过渡效果
    • 卡片动画：卡片的交互动画

特效设置​

自定义视觉特效：

  1. 在外观设置部分找到"特效设置"选项
  2. 配置以下选项：
    • 模糊效果：背景模糊效果
    • 阴影效果：元素阴影深度
    • 高亮效果：选中项目的高亮方式
    • 滚动效果：平滑滚动或即时滚动
    • 加载动画：数据加载时的动画效果

响应式设置​

移动设备适配​

针对小屏幕设备的优化：

  1. 在外观设置部分找到"响应式设置"选项
  2. 配置以下选项：
    • 移动布局：在小屏幕上使用的布局
    • 触摸优化：为触摸屏优化界面元素大小
    • 手势控制：启用/禁用触摸手势
    • 自动隐藏元素：在小屏幕上自动隐藏的元素
    • 方向适应：横屏和竖屏的不同布局

窗口大小适配​

根据窗口大小调整界面：

  1. 在响应式设置部分找到"窗口适配"选项
  2. 配置以下选项：
    • 断点设置：定义布局变化的窗口宽度
    • 列数调整：不同宽度下的列数
    • 元素缩放：元素随窗口大小缩放的行为
    • 自适应内容：内容区域的自适应行为

辅助功能​

可访问性设置​

提高应用的可访问性：

  1. 在外观设置部分找到"可访问性设置"选项
  2. 配置以下选项：
    • 高对比度模式：增强文本和背景对比
    • 放大文本：增加文本大小
    • 键盘导航：增强键盘导航支持
    • 屏幕阅读器优化：为屏幕阅读器优化界面
    • 减少动画：减少或禁用可能导致不适的动画

色盲模式​

为色盲用户优化界面：

  1. 在可访问性设置部分找到"色盲模式"选项
  2. 选择色盲类型：
    • 红绿色盲：针对红绿色盲优化颜色
    • 蓝黄色盲：针对蓝黄色盲优化颜色
    • 全色盲：使用高对比度的黑白模式
  3. 系统会自动调整界面颜色以提高可辨识度

应用外观设置​

应用主题​

将设置应用到整个应用：

  1. 配置完所有外观设置后
  2. 点击"应用设置"按钮
  3. 选择应用范围：
    • 仅当前设备：设置只应用于当前设备
    • 所有设备：通过同步将设置应用到所有设备
  4. 确认应用

重置外观设置​

恢复默认外观设置：

  1. 在外观设置页面底部
  2. 点击"重置为默认值"按钮
  3. 选择重置范围：
    • 重置所有设置：恢复所有外观设置为默认值
    • 重置特定部分：只重置选定的设置类别
  4. 确认重置操作

故障排除​

外观问题​

如果遇到外观显示问题：

  1. 尝试刷新页面
  2. 清除浏览器缓存
  3. 重置特定的外观设置
  4. 检查是否有自定义CSS冲突
  5. 尝试使用默认主题

性能问题​

如果自定义外观导致性能问题：

  1. 减少动画和特效
  2. 禁用复杂的自定义CSS
  3. 使用更简单的主题
  4. 减少同时显示的元素数量
  5. 检查浏览器扩展冲突

如果问题仍然存在，请参阅故障排除章节或联系我们的支持团队。
