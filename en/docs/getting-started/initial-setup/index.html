初始设置

安装拾光忆栈后，您需要完成一些初始设置才能开始使用。本指南将引导您完成这些设置步骤，确保拾光忆栈按照您的偏好工作。

首次启动向导​

首次启动拾光忆栈时，系统会显示设置向导：

向导将引导您完成以下基本设置：

  1. 欢迎页面 - 介绍拾光忆栈的主要功能
  2. 用户信息设置 - 设置基本用户信息
  3. 存储设置 - 配置本地和云存储选项
  4. 外观设置 - 选择主题和界面选项
  5. 权限设置 - 设置必要的浏览器权限
  6. 完成页面 - 设置完成，可以开始使用

您可以随时通过设置页面修改这些初始设置。

用户信息设置​

设置基本用户信息：

  1. 在设置向导中，进入"用户信息"步骤
  2. 输入以下信息：
    • 用户名（用于标识您的数据）
    • 显示名称（可选）
    • 电子邮件（可选，用于恢复账户）
  3. 点击"下一步"继续

存储设置​

配置数据存储选项：

本地存储设置​

  1. 在存储设置页面，找到"本地存储"部分
  2. 设置以下选项：
    • 存储位置（默认或自定义）
    • 存储限制（最大使用空间）
    • 缓存设置（缓存大小和过期时间）
  3. 点击"保存本地存储设置"

云存储设置（可选）​

如果您希望在多个设备间同步数据，可以配置云存储：

  1. 在存储设置页面，找到"云存储"部分
  2. 选择云存储提供商：
    • 华为云对象存储服务 (OBS)
    • 阿里云对象存储服务 (OSS)
    • 腾讯云对象存储 (COS)
    • 七牛云对象存储
    • Amazon S3
    • 其他兼容S3的存储服务
  3. 输入相应的访问凭证：
    • 访问密钥ID (AK)
    • 秘密访问密钥 (SK)
    • 终端节点 (Endpoint)
    • 桶名称 (Bucket)
  4. 点击"测试连接"验证配置
  5. 连接成功后点击"保存云存储设置"

同步设置​

如果您配置了云存储，还需要设置同步选项：

  1. 在设置向导中，进入"同步设置"步骤
  2. 配置以下选项：
    • 同步频率（手动、每小时、每天等）
    • 同步范围（全部数据或选择性同步）
    • 冲突解决策略（保留最新、保留两者等）
    • 网络限制（仅在Wi-Fi下同步等）
  3. 点击"保存同步设置"

外观设置​

拾光忆栈支持自定义外观，包括暗色模式、主题颜色等。

设置步骤​

  1. 在设置页面中，找到"外观设置"部分
  2. 选择您喜欢的主题模式（亮色/暗色）
  3. 调整主题颜色、字体大小等
  4. 设置是否使用紧凑模式
  5. 点击"保存"应用更改

权限设置​

拾光忆栈需要一些浏览器权限才能正常工作：

  1. 在设置向导中，进入"权限设置"步骤
  2. 查看所需权限列表：
    • 存储访问权限（保存记忆）
    • 标签页访问权限（获取页面信息）
    • 上下文菜单权限（添加右键菜单选项）
    • 通知权限（显示操作通知）
  3. 点击"授予权限"按钮
  4. 在浏览器权限请求对话框中确认授权

分类设置​

设置记忆分类结构：

  1. 在设置页面中，找到"分类管理"部分
  2. 您可以使用默认分类或创建自定义分类
  3. 创建新分类：
    • 点击"添加分类"按钮
    • 输入分类名称
    • 选择父分类（如果是子分类）
    • 点击"保存"按钮
  4. 设置默认分类（新记忆的默认分类）
  5. 点击"保存分类设置"

快捷键设置​

自定义键盘快捷键：

  1. 在设置页面中，找到"快捷键设置"部分
  2. 查看默认快捷键列表
  3. 点击要修改的快捷键旁边的"编辑"按钮
  4. 按下新的键盘组合
  5. 点击"保存"按钮应用更改

默认快捷键包括：

  • Ctrl+Shift+M（Windows/Linux）或 Cmd+Shift+M（Mac）- 保存选中内容
  • Ctrl+Shift+E（Windows/Linux）或 Cmd+Shift+E（Mac）- 编辑并保存选中内容
  • Ctrl+Shift+P（Windows/Linux）或 Cmd+Shift+P（Mac）- 打开扩展弹出窗口

隐私设置​

配置隐私相关选项：

  1. 在设置页面中，找到"隐私设置"部分
  2. 配置以下选项：
    • 数据收集（使用统计、错误报告等）
    • 自动锁定（闲置时锁定应用）
    • 隐私模式（在历史记录中隐藏操作）
    • 敏感内容处理（自动加密敏感内容）
  3. 点击"保存隐私设置"

高级设置​

配置高级选项（可选）：

  1. 在设置页面中，找到"高级设置"部分
  2. 配置以下选项：
    • 性能设置（内存使用、缓存大小等）
    • 调试模式（启用详细日志）
    • API访问（启用API接口）
    • 实验性功能（启用测试中的新功能）
  3. 点击"保存高级设置"

注意：高级设置可能会影响应用性能和稳定性。如果您不确定，建议保留默认设置。

验证设置​

完成以上设置后，您可以通过以下步骤验证设置是否生效：

  1. 点击左侧菜单的"浏览记忆"
  2. 如果您已经配置了云存储，系统会尝试从云端加载记忆
  3. 尝试添加一条新记忆，验证是否可以正常保存和同步

后续步骤​

完成初始设置后，您可以：

  1. 了解界面概览，熟悉拾光忆栈的主要功能
  2. 学习如何保存记忆
  3. 探索高级功能，如数据加密、多设备同步等

如果您在设置过程中遇到任何问题，请参阅故障排除章节或联系我们的支持团队。
