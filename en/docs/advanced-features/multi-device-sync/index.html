多设备同步

拾光忆栈支持在多个设备之间同步您的记忆，让您可以随时随地访问您的知识库。本指南将介绍如何设置和使用多设备同步功能。

同步概述​

拾光忆栈的多设备同步功能允许您：

  • 在多台计算机之间同步记忆
  • 在不同浏览器之间同步记忆
  • 在不同操作系统之间同步记忆
  • 保持所有设备上的记忆一致

同步内容包括：

  • 记忆内容（文本、图片、视频等）
  • 分类和标签结构
  • 用户设置和偏好
  • 保存的搜索和过滤器

同步方式​

拾光忆栈提供两种同步方式：

云存储同步​

使用云存储服务同步数据：

  1. 数据存储在您选择的云存储服务上
  2. 支持多种云存储提供商
  3. 您完全控制数据存储位置
  4. 提供端到端加密保护隐私

本地同步​

使用本地网络同步数据：

  1. 数据在本地网络中的设备之间直接传输
  2. 不需要互联网连接
  3. 适合对隐私要求极高的场景
  4. 需要设备在同一网络中

设置云存储同步​

支持的云存储服务​

拾光忆栈支持以下云存储服务：

  • 华为云对象存储服务 (OBS)
  • 阿里云对象存储服务 (OSS)
  • 腾讯云对象存储 (COS)
  • 七牛云对象存储
  • Amazon S3
  • Google Cloud Storage
  • Microsoft Azure Blob Storage
  • Dropbox
  • WebDAV 服务

配置云存储​

以华为云OBS为例，配置云存储的步骤：

  1. 打开设置页面
  2. 找到"同步设置"部分
  3. 点击"配置云存储"按钮
  4. 选择"华为云OBS"
  5. 输入以下信息：
    • 访问密钥ID (AK)
    • 秘密访问密钥 (SK)
    • 终端节点 (Endpoint)
    • 桶名称 (Bucket)
    • 存储路径 (可选)
  6. 点击"测试连接"验证配置
  7. 配置成功后点击"保存"按钮

启用同步​

配置云存储后，启用同步功能：

  1. 在同步设置页面，将"启用同步"开关设为开启状态
  2. 选择同步模式：
    • 自动同步 - 自动在后台同步
    • 手动同步 - 仅在点击同步按钮时同步
  3. 设置同步频率（如果选择自动同步）
  4. 点击"保存"按钮

初始同步​

首次启用同步时，您需要选择初始同步方向：

  1. 上传本地数据 - 将本地数据上传到云端
  2. 下载云端数据 - 从云端下载数据到本地
  3. 合并数据 - 合并本地和云端数据

注意：初始同步可能需要一些时间，特别是当您有大量记忆时。

设置本地同步​

配置本地同步​

设置本地同步的步骤：

  1. 打开设置页面
  2. 找到"同步设置"部分
  3. 点击"配置本地同步"按钮
  4. 选择同步方式：
    • 直接连接 - 设备之间直接连接
    • 本地服务器 - 通过本地服务器中转
  5. 如果选择直接连接：
    • 在主设备上生成配对码
    • 在其他设备上输入配对码
  6. 如果选择本地服务器：
    • 设置服务器地址和端口
    • 设置访问密码
  7. 点击"保存"按钮

设备管理​

管理同步设备的步骤：

  1. 在同步设置页面，找到"设备管理"部分
  2. 查看所有已连接的设备
  3. 为设备命名以便识别
  4. 设置设备优先级
  5. 移除不再使用的设备

同步设置​

同步范围​

您可以选择要同步的内容：

  1. 在同步设置页面，找到"同步范围"部分
  2. 选择要同步的内容类型：
    • 记忆内容 - 所有保存的记忆
    • 分类和标签 - 分类结构和标签
    • 用户设置 - 应用设置和偏好
    • 搜索和过滤器 - 保存的搜索和过滤条件
  3. 点击"保存"按钮

同步排除​

您可以排除特定内容不进行同步：

  1. 在同步设置页面，找到"同步排除"部分
  2. 设置排除规则：
    • 排除特定分类
    • 排除特定标签
    • 排除特定时间段的记忆
    • 排除特定类型的内容
  3. 点击"保存"按钮

同步频率​

设置自动同步的频率：

  1. 在同步设置页面，找到"同步频率"部分
  2. 选择同步频率：
    • 实时同步 - 内容变更时立即同步
    • 定时同步 - 按设定的时间间隔同步
    • 启动时同步 - 应用启动时同步
    • 关闭时同步 - 应用关闭时同步
  3. 如果选择定时同步，设置时间间隔
  4. 点击"保存"按钮

网络设置​

配置同步的网络设置：

  1. 在同步设置页面，找到"网络设置"部分
  2. 配置以下选项：
    • 仅在Wi-Fi网络下同步 - 避免使用移动数据
    • 带宽限制 - 限制同步使用的带宽
    • 代理设置 - 配置网络代理
  3. 点击"保存"按钮

冲突解决​

当多个设备修改同一内容时，可能会发生冲突。拾光忆栈提供以下冲突解决策略：

自动冲突解决​

配置自动冲突解决策略：

  1. 在同步设置页面，找到"冲突解决"部分
  2. 选择默认策略：
    • 保留最新版本 - 使用最近修改的版本
    • 保留所有版本 - 保留所有冲突版本
    • 设备优先级 - 按设备优先级决定
    • 总是询问 - 冲突时手动选择
  3. 点击"保存"按钮

手动冲突解决​

当设置为"总是询问"或自动解决失败时：

  1. 系统会显示冲突解决对话框
  2. 显示冲突内容的不同版本
  3. 您可以选择保留哪个版本，或合并内容
  4. 选择"应用到所有冲突"可以批量处理多个冲突

同步状态和历史​

查看同步状态​

查看当前同步状态：

  1. 点击工具栏中的同步图标
  2. 查看同步状态信息：
    • 上次同步时间
    • 当前同步进度
    • 待同步项目数量
    • 同步错误（如果有）

同步历史​

查看同步历史记录：

  1. 在同步设置页面，找到"同步历史"部分
  2. 查看详细的同步历史记录：
    • 同步时间
    • 同步内容摘要
    • 同步结果
    • 错误详情（如果有）
  3. 点击特定记录可查看详细信息

数据安全​

端到端加密​

启用端到端加密保护同步数据：

  1. 在同步设置页面，找到"安全设置"部分
  2. 启用"端到端加密"选项
  3. 设置加密密码或密钥
  4. 选择加密算法和强度
  5. 点击"保存"按钮

重要：请妥善保管您的加密密码。如果忘记密码，将无法恢复加密的数据。

传输加密​

所有同步数据在传输过程中都会加密：

  1. 使用TLS/SSL加密网络传输
  2. 支持自定义证书（高级选项）
  3. 可以设置额外的传输层安全选项

高级同步功能​

选择性同步​

设置选择性同步特定内容：

  1. 在同步设置页面，找到"选择性同步"部分
  2. 选择要同步的特定分类或标签
  3. 设置内容筛选条件
  4. 点击"保存"按钮

同步计划​

创建自定义同步计划：

  1. 在同步设置页面，找到"同步计划"部分
  2. 点击"创建新计划"按钮
  3. 设置同步时间和频率
  4. 设置同步内容范围
  5. 设置网络条件
  6. 点击"保存"按钮

同步通知​

配置同步通知：

  1. 在同步设置页面，找到"通知设置"部分
  2. 选择需要通知的事件：
    • 同步完成
    • 同步错误
    • 冲突发生
    • 存储空间不足
  3. 设置通知方式
  4. 点击"保存"按钮

故障排除​

同步失败​

如果同步失败，请尝试以下解决方法：

  1. 检查网络连接
  2. 验证云存储配置是否正确
  3. 检查存储空间是否充足
  4. 查看详细错误日志
  5. 尝试手动触发同步

数据不一致​

如果设备间数据不一致：

  1. 强制完全同步：
    • 在同步设置页面点击"强制完全同步"按钮
    • 选择同步方向
    • 等待同步完成
  2. 检查同步排除设置
  3. 检查是否有未解决的冲突

同步速度慢​

如果同步速度很慢：

  1. 检查网络连接质量
  2. 减少同步数据量
  3. 调整同步频率
  4. 检查带宽限制设置
  5. 尝试更换云存储服务

如果问题仍然存在，请参阅故障排除章节或联系我们的支持团队。
