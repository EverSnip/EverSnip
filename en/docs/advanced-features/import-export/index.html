数据导入导出

拾光忆栈提供全面的数据导入导出功能，让您可以备份数据、迁移到新设备或与其他应用交换数据。本指南将介绍如何使用这些功能。

导出数据​

完整导出​

导出所有数据的步骤：

  1. 打开设置页面
  2. 找到"数据管理"部分
  3. 点击"导出数据"按钮
  4. 选择"完整导出"
  5. 选择导出格式：
    • 拾光忆栈格式 (.mke) - 完整备份，包含所有数据和设置
    • JSON格式 (.json) - 通用格式，便于其他应用读取
    • HTML格式 (.html) - 可在浏览器中查看的格式
    • Markdown格式 (.md) - 适合文本编辑器查看
  6. 设置导出选项：
    • 是否包含媒体文件
    • 是否加密导出文件
    • 是否分割大文件
  7. 点击"导出"按钮
  8. 选择保存位置

选择性导出​

导出部分数据的步骤：

  1. 打开设置页面
  2. 找到"数据管理"部分
  3. 点击"导出数据"按钮
  4. 选择"选择性导出"
  5. 选择要导出的内容：
    • 特定分类
    • 特定标签
    • 日期范围
    • 内容类型（文本、图片、视频等）
  6. 选择导出格式和选项
  7. 点击"导出"按钮
  8. 选择保存位置

批量导出​

从浏览页面导出选定记忆：

  1. 在浏览记忆页面，选择要导出的记忆
  2. 点击"更多"菜单
  3. 选择"导出选中项"
  4. 选择导出格式和选项
  5. 点击"导出"按钮
  6. 选择保存位置

导出格式详解​

拾光忆栈格式 (.mke)​

  • 专有格式，保留所有数据和关系
  • 包含记忆内容、元数据、分类、标签、设置等
  • 支持增量备份
  • 可以加密保护
  • 最适合完整备份和恢复

JSON格式 (.json)​

  • 通用格式，易于其他应用读取
  • 保留数据结构和关系
  • 包含记忆内容和元数据
  • 可以选择是否包含媒体文件
  • 适合数据迁移和分析

HTML格式 (.html)​

  • 可在任何浏览器中查看
  • 保留格式和样式
  • 包含交互式导航和搜索功能
  • 可以作为独立网页或网站
  • 适合分享和归档

Markdown格式 (.md)​

  • 纯文本格式，适合任何编辑器
  • 保留基本格式（标题、列表、链接等）
  • 可以包含图片引用
  • 轻量级，易于编辑
  • 适合文档化和知识库构建

CSV格式 (.csv)​

  • 表格格式，适合电子表格应用
  • 包含基本元数据和内容摘要
  • 易于排序和过滤
  • 不包含复杂格式和媒体
  • 适合数据分析和批量处理

导出加密​

保护导出数据的安全：

  1. 在导出选项中启用"加密导出文件"
  2. 选择加密方式：
    • 使用现有主密码
    • 设置新密码
    • 使用密钥文件
  3. 设置加密强度
  4. 导出的文件将被加密，需要密码才能导入

重要：请妥善保管导出文件的密码。如果忘记密码，将无法恢复加密的导出文件。

自动导出​

设置定期自动导出：

  1. 在数据管理设置中找到"自动导出"部分
  2. 启用"定期自动导出"
  3. 设置导出频率（每天、每周、每月）
  4. 选择导出格式和选项
  5. 设置导出位置（本地文件夹或云存储）
  6. 设置保留的备份数量
  7. 点击"保存"按钮

导入数据​

从备份导入​

从拾光忆栈备份文件导入：

  1. 打开设置页面
  2. 找到"数据管理"部分
  3. 点击"导入数据"按钮
  4. 选择"从备份导入"
  5. 选择备份文件（.mke格式）
  6. 如果文件已加密，输入密码
  7. 选择导入选项：
    • 完全替换 - 清除现有数据，完全使用备份数据
    • 合并 - 将备份数据与现有数据合并
    • 仅导入新项目 - 只导入现有数据中不存在的项目
  8. 点击"导入"按钮
  9. 等待导入完成

从其他格式导入​

从其他格式文件导入：

  1. 打开设置页面
  2. 找到"数据管理"部分
  3. 点击"导入数据"按钮
  4. 选择"从文件导入"
  5. 选择文件类型：
    • JSON文件
    • HTML文件
    • Markdown文件
    • CSV文件
    • 文本文件
  6. 选择文件
  7. 配置导入映射（如果需要）
  8. 选择导入选项
  9. 点击"导入"按钮

从剪贴板导入​

快速导入剪贴板内容：

  1. 复制要导入的内容
  2. 打开拾光忆栈
  3. 点击工具栏中的"从剪贴板导入"按钮
  4. 预览内容并编辑（如需要）
  5. 设置分类、标签和其他元数据
  6. 点击"导入"按钮

批量导入​

导入文件夹中的多个文件：

  1. 在设置页面，找到"数据管理"部分
  2. 点击"导入数据"按钮
  3. 选择"批量导入"
  4. 选择文件夹或多个文件
  5. 设置导入规则：
    • 文件命名规则
    • 文件夹结构映射
    • 元数据提取规则
  6. 预览导入结果
  7. 点击"导入"按钮

从其他服务导入​

拾光忆栈支持从其他服务导入数据：

从Evernote导入​

  1. 在导入对话框中选择"从其他服务导入"
  2. 选择"Evernote"
  3. 上传Evernote导出的ENEX文件
  4. 配置导入选项
  5. 点击"导入"按钮

从OneNote导入​

  1. 在导入对话框中选择"从其他服务导入"
  2. 选择"OneNote"
  3. 上传OneNote导出的文件
  4. 配置导入选项
  5. 点击"导入"按钮

从印象笔记导入​

  1. 在导入对话框中选择"从其他服务导入"
  2. 选择"印象笔记"
  3. 上传印象笔记导出的文件
  4. 配置导入选项
  5. 点击"导入"按钮

自动备份​

拾光忆栈可以自动备份您的数据：

配置自动备份​

  1. 打开设置页面
  2. 找到"备份设置"部分
  3. 启用"自动备份"选项
  4. 配置备份频率：
    • 每天
    • 每周
    • 每月
  5. 选择备份位置：
    • 云存储
    • 本地存储
  6. 设置保留的备份数量
  7. 保存设置

管理备份​

查看和管理现有备份：

  1. 打开设置页面
  2. 找到"备份设置"部分
  3. 点击"管理备份"按钮
  4. 在备份列表中，您可以：
    • 查看备份详情
    • 下载备份
    • 恢复备份
    • 删除备份

恢复备份​

从备份恢复数据：

  1. 在备份管理页面，找到要恢复的备份
  2. 点击"恢复"按钮
  3. 选择恢复选项：
    • 完全恢复
    • 选择性恢复
  4. 确认恢复操作
  5. 等待恢复完成

数据迁移​

设备迁移​

将数据迁移到新设备：

  1. 在旧设备上导出完整备份
  2. 将备份文件传输到新设备
  3. 在新设备上安装拾光忆栈
  4. 导入备份文件
  5. 验证数据完整性

浏览器迁移​

在不同浏览器之间迁移：

  1. 在原浏览器中导出完整备份
  2. 在新浏览器中安装拾光忆栈
  3. 导入备份文件
  4. 验证设置和数据

高级导入导出功能​

增量备份​

只备份自上次备份以来的变更：

  1. 在备份设置中启用"增量备份"
  2. 设置完整备份频率（如每月一次）
  3. 设置增量备份频率（如每天一次）
  4. 系统会自动管理备份链

差异导出​

导出两个时间点之间的数据变化：

  1. 在导出对话框中选择"差异导出"
  2. 选择基准时间点（如上次备份时间）
  3. 系统会只导出自该时间点以来的新增和修改内容
  4. 可用于创建更新包或跟踪变化

导出模板​

创建自定义导出模板：

  1. 在导出设置中点击"创建导出模板"
  2. 设置模板名称
  3. 配置导出选项和过滤条件
  4. 保存模板
  5. 之后可以直接使用模板快速导出

API导出​

通过API自动化导出过程：

  1. 在设置中启用API访问
  2. 生成API密钥
  3. 使用API端点触发导出
  4. 可以与其他自动化工具集成

导入导出安全性​

数据隐私​

保护导出数据的隐私：

  1. 使用密码加密导出文件
  2. 避免在公共设备上保存未加密的导出文件
  3. 定期删除不需要的旧备份
  4. 使用安全的传输方式分享备份文件

导出验证​

验证导出文件的完整性：

  1. 导出完成后，系统会生成校验和
  2. 可以使用校验和验证文件是否损坏
  3. 导入前会自动验证文件完整性
  4. 如果文件损坏，系统会提示错误

故障排除​

导出失败​

如果导出失败，请尝试以下解决方法：

  1. 检查存储空间是否充足
  2. 尝试导出较小的数据集
  3. 使用分割选项处理大文件
  4. 检查文件权限
  5. 尝试不同的导出格式

导入错误​

如果导入出现错误：

  1. 确保文件格式正确
  2. 检查文件是否损坏
  3. 如果文件已加密，确保密码正确
  4. 尝试导入较小的数据集
  5. 检查应用版本兼容性

如果问题仍然存在，请参阅故障排除章节或联系我们的支持团队。
