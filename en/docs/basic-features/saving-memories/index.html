保存记忆

拾光忆栈提供多种方式从网页中保存内容，包括文本、图片和视频。本指南将介绍如何使用这些功能保存您的记忆。

快速保存​

最简单的保存方式是使用快速保存功能：

  1. 在网页上选中您想要保存的文本
  2. 按下快捷键 Ctrl+Shift+M（Windows/Linux）或 Cmd+Shift+M（Mac）
  3. 系统会使用默认设置保存选中的内容

使用右键菜单保存​

您也可以通过右键菜单保存内容：

  1. 在网页上选中您想要保存的文本
  2. 右键点击选中的内容
  3. 在弹出的菜单中选择"保存到拾光忆栈"
  4. 在弹出的对话框中编辑标题、添加标签和选择分类
  5. 点击"保存"按钮

使用扩展弹出窗口保存​

通过扩展弹出窗口，您可以保存当前页面或自定义内容：

  1. 点击Chrome工具栏中的拾光忆栈图标
  2. 在弹出窗口中，选择"保存当前页面"或"添加新记忆"
  3. 编辑记忆的详细信息
  4. 点击"保存"按钮

文本记忆​

保存文本记忆时，您可以：

保存选中文本​

  1. 在网页上选中文本
  2. 使用快捷键、右键菜单或扩展弹出窗口保存
  3. 系统会保留原始格式，包括链接、列表和表格

编辑保存的文本​

在保存前，您可以编辑文本内容：

  1. 在保存对话框中，编辑文本内容
  2. 使用富文本编辑器调整格式
  3. 添加自己的注释或标记重点内容

保存整个文章​

您可以保存整篇文章：

  1. 不选中任何内容，直接使用快捷键或右键菜单
  2. 系统会自动检测页面的主要内容
  3. 在保存对话框中确认或编辑内容

图片记忆​

拾光忆栈允许您保存网页中的图片：

保存单张图片​

  1. 右键点击网页中的图片
  2. 选择"保存图片到拾光忆栈"
  3. 在弹出的对话框中编辑标题和描述
  4. 点击"保存"按钮

保存带有说明的图片​

您可以为图片添加说明：

  1. 保存图片时，在描述字段中添加说明
  2. 您也可以为图片添加标签，方便后续查找
  3. 选择适当的分类存储图片

图片编辑选项​

保存前，您可以简单编辑图片：

  1. 在保存对话框中，点击"编辑图片"按钮
  2. 使用内置工具裁剪、旋转或调整图片
  3. 完成编辑后点击"应用"按钮

视频记忆​

您也可以保存网页中的视频内容：

保存视频引用​

  1. 右键点击网页中的视频
  2. 选择"保存视频到拾光忆栈"
  3. 在弹出的对话框中编辑标题和描述
  4. 点击"保存"按钮

保存视频截图​

您可以保存视频的特定帧：

  1. 在视频播放到您想要保存的画面时暂停
  2. 右键点击视频，选择"保存视频截图到拾光忆栈"
  3. 编辑截图的标题和描述
  4. 点击"保存"按钮

保存视频时间点​

您可以保存视频的特定时间点：

  1. 在视频播放到您想要标记的时间点时
  2. 使用快捷键或右键菜单保存
  3. 系统会自动记录当前的时间点
  4. 之后查看记忆时，可以直接跳转到该时间点

保存网页链接​

除了内容外，您还可以保存网页链接：

  1. 右键点击页面空白处或链接
  2. 选择"保存链接到拾光忆栈"
  3. 系统会自动提取页面标题和描述
  4. 编辑信息并保存

批量保存​

在某些情况下，您可能需要批量保存内容：

保存多张图片​

  1. 按住Ctrl键（Windows/Linux）或Cmd键（Mac）选择多张图片
  2. 右键点击，选择"批量保存到拾光忆栈"
  3. 在弹出的对话框中设置共同的标签和分类
  4. 点击"保存全部"按钮

保存页面中的所有链接​

  1. 点击扩展图标，选择"高级功能"
  2. 选择"提取页面链接"
  3. 选择您想要保存的链接
  4. 点击"保存选中链接"按钮

自动保存设置​

您可以配置自动保存规则：

  1. 打开设置页面
  2. 找到"自动保存规则"部分
  3. 创建新规则，指定网站、内容类型和保存条件
  4. 启用规则

例如，您可以设置规则自动保存特定网站的所有文章或图片。

保存选项​

保存记忆时，您可以设置以下选项：

基本选项​

  • 标题：记忆的标题（默认使用选中文本的前几个字或图片/视频的原始标题）
  • 内容：记忆的主要内容
  • 来源URL：内容的原始网址（自动填充）
  • 来源网站：内容的来源网站名称（自动填充）

组织选项​

  • 分类：选择一个分类存储记忆
  • 标签：添加标签帮助组织和查找记忆
  • 收藏：标记为收藏的记忆会显示在收藏夹中

高级选项​

  • 重要性：设置记忆的重要程度
  • 提醒：设置提醒，在特定时间回顾记忆
  • 加密：启用端到端加密保护敏感记忆

保存模板​

如果您经常保存特定类型的内容，可以创建保存模板：

  1. 打开设置页面
  2. 找到"保存模板"部分
  3. 点击"创建新模板"按钮
  4. 设置模板的默认选项
  5. 保存模板

之后，您可以在保存内容时选择应用这些模板。

保存快捷键​

以下是保存记忆的常用快捷键：

  • Ctrl+Shift+M（Windows/Linux）或 Cmd+Shift+M（Mac）：保存选中内容
  • Ctrl+Shift+E（Windows/Linux）或 Cmd+Shift+E（Mac）：编辑并保存选中内容
  • Ctrl+Shift+P（Windows/Linux）或 Cmd+Shift+P（Mac）：打开扩展弹出窗口

您可以在设置页面自定义这些快捷键。

故障排除​

如果您在保存记忆时遇到问题，请尝试以下解决方法：

无法选择内容​

某些网站可能会阻止内容选择。尝试：

  1. 禁用网站的复制保护（如果您有权限）
  2. 使用"阅读模式"查看内容
  3. 使用扩展弹出窗口手动添加内容

格式丢失​

如果保存的内容丢失了原始格式，尝试：

  1. 在设置中启用"保留原始格式"选项
  2. 使用"高级保存"选项
  3. 保存为PDF或截图

无法保存特定网站的内容​

某些网站可能有特殊保护。尝试：

  1. 检查网站的使用条款
  2. 使用截图功能代替直接保存
  3. 手动复制内容并使用扩展弹出窗口保存

如果问题仍然存在，请参阅故障排除章节或联系我们的支持团队。
