搜索与过滤

拾光忆栈提供强大的搜索和过滤功能，帮助您在大量记忆中快速找到所需信息。本指南将详细介绍如何使用这些功能。

基本搜索​

快速搜索​

最简单的搜索方式是使用顶部工具栏中的搜索框：

  1. 在浏览记忆页面顶部的搜索框中输入关键词
  2. 按回车键或点击搜索图标
  3. 系统会显示包含该关键词的所有记忆

搜索范围​

默认情况下，搜索会在以下字段中查找匹配项：

  • 记忆标题
  • 记忆内容
  • 标签
  • 来源URL
  • 来源网站名称

实时搜索​

启用实时搜索后，系统会在您输入时自动显示搜索结果：

  1. 在设置中启用"实时搜索"选项
  2. 在搜索框中输入时，结果会随着输入自动更新
  3. 无需按回车即可看到匹配结果

高级搜索​

搜索语法​

拾光忆栈支持高级搜索语法，可以构建复杂的搜索查询：

  • 引号：使用引号搜索精确短语，如 "机器学习"
  • 加号：使用加号指定必须包含的词，如 +Python +教程
  • 减号：使用减号排除包含特定词的结果，如 机器学习 -深度
  • 字段限定：指定搜索特定字段，如 title:Python 或 content:算法
  • OR 操作符：使用 OR 搜索多个可能的词，如 Python OR Java
  • AND 操作符：使用 AND 要求同时包含多个词，如 Python AND 数据科学
  • 通配符：使用 表示任意字符，如 `程序语言`

高级搜索面板​

使用高级搜索面板构建复杂查询：

  1. 点击搜索框旁边的"高级"按钮
  2. 在弹出的面板中设置以下条件：
    • 包含全部关键词：结果必须包含所有指定的关键词
    • 包含任意关键词：结果可以包含任意一个指定的关键词
    • 不包含关键词：结果不能包含指定的关键词
    • 精确匹配短语：结果必须包含完全一致的短语
  3. 设置搜索范围（标题、内容、标签等）
  4. 设置时间范围
  5. 点击"搜索"按钮

过滤功能​

按分类过滤​

通过分类过滤记忆：

  1. 在浏览记忆页面左侧的分类面板中点击一个分类
  2. 系统会显示该分类下的所有记忆
  3. 点击分类名称旁边的"+"图标可以展开子分类
  4. 点击"全部"显示所有分类的记忆

按标签过滤​

通过标签过滤记忆：

  1. 在浏览记忆页面左侧的标签面板中点击一个标签
  2. 系统会显示带有该标签的所有记忆
  3. 点击多个标签可以组合过滤（显示同时包含所有选中标签的记忆）
  4. 点击已选标签可以取消选择

按日期过滤​

通过日期范围过滤记忆：

  1. 点击工具栏中的"日期"下拉菜单
  2. 选择预设日期范围（今天、本周、本月、今年）
  3. 或选择"自定义范围"并设置开始和结束日期
  4. 系统会显示在选定日期范围内创建的记忆

按类型过滤​

通过内容类型过滤记忆：

  1. 点击工具栏中的"类型"下拉菜单
  2. 选择内容类型（文本、图片、视频、链接等）
  3. 系统会显示选定类型的记忆

按来源过滤​

通过来源网站过滤记忆：

  1. 点击工具栏中的"来源"下拉菜单
  2. 选择一个来源网站
  3. 系统会显示来自该网站的所有记忆

组合过滤​

您可以组合多个过滤条件：

  1. 先选择一个分类
  2. 再选择一个或多个标签
  3. 设置日期范围
  4. 选择内容类型
  5. 系统会显示同时满足所有条件的记忆

搜索结果​

排序搜索结果​

您可以按不同条件排序搜索结果：

  1. 在搜索结果页面点击"排序"下拉菜单
  2. 选择排序条件：
    • 相关性 - 按匹配度排序（默认）
    • 最新添加 - 按添加时间从新到旧排序
    • 最早添加 - 按添加时间从旧到新排序
    • 标题 A-Z - 按标题字母顺序排序
    • 标题 Z-A - 按标题字母逆序排序

结果高亮​

搜索结果中会高亮显示匹配的关键词：

  • 在记忆标题中高亮显示
  • 在记忆内容摘要中高亮显示
  • 在记忆详情中高亮显示

结果统计​

搜索结果页面会显示统计信息：

  • 匹配记忆的总数
  • 搜索用时
  • 当前显示的结果范围

保存搜索​

您可以保存常用的搜索条件，方便日后快速访问：

保存当前搜索​

  1. 设置所需的搜索和过滤条件
  2. 点击搜索框旁边的"保存"图标
  3. 输入搜索名称
  4. 点击"保存"按钮

使用保存的搜索​

  1. 点击搜索框旁边的"已保存搜索"下拉菜单
  2. 选择一个保存的搜索
  3. 系统会自动应用相应的搜索和过滤条件

管理保存的搜索​

  1. 点击"已保存搜索"下拉菜单
  2. 选择"管理已保存搜索"
  3. 在弹出的对话框中，您可以：
    • 重命名保存的搜索
    • 更新搜索条件
    • 删除不再需要的搜索

搜索历史​

拾光忆栈会自动记录您的搜索历史：

查看搜索历史​

  1. 点击搜索框，会显示最近的搜索历史
  2. 点击历史记录可以快速重复该搜索

清除搜索历史​

  1. 点击搜索框旁边的"更多"图标
  2. 选择"清除搜索历史"
  3. 确认清除

全文搜索索引​

拾光忆栈使用全文搜索索引提高搜索性能：

索引状态​

您可以在设置中查看索引状态：

  1. 打开设置页面
  2. 找到"搜索设置"部分
  3. 查看索引状态、上次更新时间和索引大小

重建索引​

如果搜索结果不准确或索引损坏，您可以重建索引：

  1. 打开设置页面
  2. 找到"搜索设置"部分
  3. 点击"重建搜索索引"按钮
  4. 等待索引重建完成

注意：索引重建可能需要一些时间，特别是当您有大量记忆时。

搜索设置​

您可以在设置页面自定义搜索行为：

  1. 打开设置页面
  2. 找到"搜索设置"部分
  3. 配置以下选项：
    • 实时搜索：启用/禁用输入时自动搜索
    • 搜索结果数量：每页显示的结果数量
    • 高亮匹配项：是否高亮显示匹配的关键词
    • 模糊匹配：启用/禁用模糊匹配
    • 搜索历史：启用/禁用搜索历史记录
    • 默认排序：设置默认的结果排序方式

高级搜索技巧​

使用正则表达式​

启用正则表达式搜索：

  1. 在设置中启用"正则表达式搜索"选项
  2. 在搜索框中使用 / 包围正则表达式，如 /\d{4}-\d{2}-\d{2}/
  3. 这可以用于搜索特定格式的文本，如日期、电话号码等

搜索图片内容​

拾光忆栈支持搜索图片中的文本：

  1. 在设置中启用"图片文本识别"选项
  2. 系统会使用OCR技术识别图片中的文本
  3. 搜索时会包含图片中的文本内容

语义搜索​

使用语义搜索找到概念相关的内容：

  1. 在设置中启用"语义搜索"选项
  2. 使用 ~ 前缀启用特定搜索的语义模式，如 ~机器学习
  3. 系统会返回与搜索概念语义相关的结果，而不仅仅是包含关键词的结果

搜索快捷键​

以下是搜索相关的常用快捷键：

  • Ctrl+F（Windows/Linux）或 Cmd+F（Mac）：聚焦到搜索框
  • Ctrl+Shift+F（Windows/Linux）或 Cmd+Shift+F（Mac）：打开高级搜索面板
  • Esc：清除搜索框内容
  • Enter：执行搜索
  • Alt+S：保存当前搜索

您可以在设置页面自定义这些快捷键。

故障排除​

如果您在使用搜索和过滤功能时遇到问题，请尝试以下解决方法：

搜索结果不准确​

如果搜索结果不准确或缺失：

  1. 尝试重建搜索索引
  2. 检查搜索语法是否正确
  3. 尝试使用不同的关键词或更简单的查询

过滤器不工作​

如果过滤器不正常工作：

  1. 尝试清除所有过滤条件，然后重新应用
  2. 检查是否有冲突的过滤条件
  3. 刷新页面后重试

搜索速度慢​

如果搜索速度很慢：

  1. 减少记忆数量或优化存储
  2. 在设置中调整索引配置
  3. 使用更具体的搜索条件

如果问题仍然存在，请参阅故障排除章节或联系我们的支持团队。
